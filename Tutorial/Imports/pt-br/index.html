lows( df: pd.DataFrame )

size( df: pd.DataFrame )

mFormatNumber 
* Métodos para formatar números com casa decimal muito grande

scientificNotation( number ) -> bool
- Retorna verdadeiro se o número estiver em notação científica.

formatNumber( number ) -> str
- Faz uso do método scientificNotation
- Número float 10.1221212, converte para 10.122, limita para 3 casas decimais.  
- Número float 0.0000052505211455 limita para 10 casas decimais.

decimalFormat( number ) -> str
- Método isolado para limitar número float em 3 casas decimais.
mCollection 
* Conjunto de métodos úteis para extrair algum valor específico  

selectDay(
   df:     pd.DataFrame, 
   numDay: int = 1
) -> int
- Retorna o candle de abertura de X dias anteriores escolhido.

indexByDate(
    df:   pd.DataFrame,
    date: str
) -> int
- Retorna o index de uma data específica, ex: "2024-05-20 09:00" ou a data através da série 'dates' devidamente convertido para string.
    
maxPrice(
    df:         pd.DataFrame, 
    startIndex: int, 
    endIndex:   int
) -> float
- Retorna o preço máximo de um determinado range de candles.

minPrice(
    df:         pd.DataFrame, 
    startIndex: int, 
    endIndex:   int
) -> float
- Retorna o preço mínimo de um determinado range de candles.
    
indexMaxPrice(
    df:         pd.DataFrame, 
    startIndex: int, 
    endIndex:   int
) -> int
- Retorna o index do preço máximo de um range.
    
indexMinPrice(
    df:         pd.DataFrame, 
    startIndex: int, 
    endIndex:   int
) -> int
- Retorna o index do preço mínimo de um range.

shiftDateHours(
    date:  pd.Timestamp, 
    shift: int
) -> pd.Timestamp
- Passando uma data desloca para + horas ou - horas.

shiftDateMinutes(
    date:  pd.Timestamp, 
    shift: int
) -> pd.Timestamp
- Passando uma data desloca para + minutos ou - minutos.

pdDatetime(
    date: str
) -> pd.Timestamp
- Retorna uma data no formato pandas datetime.

hexcolorToRgba(
    hexColor: str, 
    alpha:  float
) -> str
- Converte cor hexadecimal para rgba

valuesInList(
    lstSrc:          list, 
    lstDefaultValue: list
) -> list
- Utilizado em inputs para garantir valores a lista.

libs 
* Contém métodos para indicador

MA(
    srcPrice: list, 
    period:   int, 
    avgType:  str
)
- Retorna uma lista de médias.
- avgType: 'sma', 'ema', 'wma', 'smma', 'zlema'
* A lista retornada terá o seu tamanho menor que a lista original enviada por parâmetro.
Lembrar deste detalhe ao iterar a lista de médias em um loop, observe os exemplos nos indicadores do projeto.
    
VWAP(
    lstPrice:  list,
    lstVolume: list,
    startIndex: int 
)
- A partir do parâmetro startIndex será iniciado o cálculo da vwap.
- Retorna uma lista com preços do cálculo da vwap.

STDDEV(
    srcPrice: list, 
    period:   int
)
- Retorna uma lista de valores do 'Standard deviation (STDDEV)'
    
ATR(
    highList:  list,
    lowList:   list,
    closeList: list, 
    period:     int
)
- Retorna uma lista de valores 'Average True Range (ATR)'
mWriteLog e mConsole 

* mWriteLog - Responsável por gravar em log erros ocorridos nos indicadores.
    Não tem utilidade para o usuário, é um import automático.
    
* mConsole  - Utilizado dentro de indicadores e inputs para verificar valores de variáveis e etc.
    Passar como string console(f'variavel {var}')
